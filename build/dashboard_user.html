Votre tableau de bord personnel

Bienvenue sur votre espace PAGOC MOOC

Une fois connecté, vous arrivez sur votre tableau de bord : c’est votre centre de command personnel où vous pouvez suivre votre progression, accéder rapidement à vos cours et voir toutes vos activités d’apprentissage.

Ce que vous voyez sur votre tableau de bord

1. Message de bienvenue personnalisé - « Bonjour [Votre Prénom] » : Un accueil chaleureux - Vos points d’expérience (XP) : Vos points cumulés - Votre niveau : Votre progression dans la plateforme - Vos badges : Récompenses que vous avez gagnées

2. Statistiques rapides - Modules commencés : Combien de cours vous avez lancés - Modules complétés : Ceux que vous avez terminés avec succès - Temps total d’apprentissage : Le temps que vous avez investi - Score moyen : Votre performance générale aux quiz

3. Section « Continuer à apprendre » - Votre module actuel : Le cours que vous suiviez en dernier - Barre de progression : Où vous en êtes dans ce module - Pourcentage de complétion : Ex: « 65% terminé » - Bouton « Reprendre » : Pour continuer exactement où vous vous étiez

4. Activité de la semaine - Graphique des 7 derniers jours : Votre activité récente - Temps passé chaque jour : Vos sessions d’apprentissage - Modules consultés : Ce que vous avez étudié récemment - Quiz passés : Vos tests et vos scores

Comment utiliser votre tableau de bord

Pour reprendre votre formation : 1. Regardez la section « Continuer à apprendre » 2. Cliquez sur « Reprendre » pour continuer votre dernier cours 3. Vous serez redirigé exactement où vous vous étiez

Pour voir votre progression globale : 1. Consultez vos statistiques en haut de la page 2. Cliquez sur « Mon Apprentissage » pour plus de détails 3. Explorez chaque module pour voir votre avancement

Pour accéder rapidement aux cours : 1. Utilisez le menu « Cours » dans la barre de navigation 2. Parcourez le catalogue des 8 modules 3. Commencez un nouveau cours en un clic

Le menu de navigation

Menu principal (en haut de la page) : - Dashboard : Votre page actuelle (tableau de bord) - Cours : Tous les modules disponibles - Mon Apprentissage : Votre progression détaillée - À Propos : Informations sur le projet PAGOC - Contact : Nous contacter si besoin

Options utilisateur (à droite) : - Drapeaux : Changer la langue (Français/Anglais) - Votre profil : Modifier vos informations - Notifications : Voir vos alertes - Déconnexion : Quitter la plateforme

Vos statistiques expliquées

Les points d’expérience (XP) : - Vous gagnez des XP en regardant des vidéos, en faisant des quiz - Plus vous étudiez, plus vous gagnez de points - Les XP vous font passer au niveau supérieur

Les niveaux : - Niveau 1 : Débutant (0-100 XP) - Niveau 2 : Intermédiaire (101-500 XP) - Niveau 3 : Avancé (501+ XP) - Chaque niveau débloque des nouvelles fonctionnalités

Les badges : - Badge « Premiers pas » : Après votre premier cours - Badge « Persévérant » : 7 jours d’affilée - Badge « Expert » : Score > 90% à un quiz - Badge « Communauté » : Aide aux autres apprenants

Comment interpréter votre progression

Barre de progression d’un module : - Vert : Vous êtes bien avancé (> 60%) - Orange : Vous êtes en milieu de parcours (30-60%) - Rouge : Vous venez de commencer (< 30%)

Temps d’apprentissage : - Optimal : 2-3 heures par semaine régulières - Insuffisant : Moins d’1 heure par semaine - Idéal : 4-5 heures par semaine pour progression rapide

Scores aux quiz : - Excellent : 90-100% : Maîtrise du sujet - Bien : 70-89% : Bonnes connaissances - À améliorer : 50-69% : Révision nécessaire - Insuffisant : < 50% : Recommencer le module

Actions rapides depuis votre tableau de bord

« Continuer à apprendre » - Reprendre exactement votre dernière session - Idéal pour continuer sans perdre de temps - Vous replace au bon endroit dans la vidéo

« Voir tous les cours » - Accéder au catalogue complet - Découvrir de nouveaux modules - Comparer les différents thèmes

« Mon profil » - Modifier vos informations personnelles - Vos certificats obtenus - Vos préférences d’apprentissage

« Notifications » - Nouveaux messages des formateurs - Rappels de formation - Annonces importantes

Personnaliser votre expérience

Changer votre photo de profil : 1. Cliquez sur votre avatar en haut à droite 2. Choisissez « Mon profil » 3. Cliquez sur « Changer la photo » 4. Sélectionnez une image et validez

Définir vos objectifs : 1. Allez dans « Mon Apprentissage » 2. Cliquez sur « Mes objectifs » 3. Fixez-vous un but (ex: 1 module par mois) 4. Suivez votre progression vers cet objectif

Choisir vos préférences : - Langue de l’interface - Fréquence des notifications - Thème clair ou sombre - Taille des polices

Conseils pour utiliser efficacement votre tableau de bord

1. Consultez-le chaque jour - Prenez l’habitude de vous connecter régulièrement - Vérifiez votre progression quotidienne - Utilisez « Continuer à apprendre » pour ne pas perdre le fil

2. Fixez-vous des objectifs réalistes - Visez 2-3 sessions par semaine - Notez vos disponibilités dans votre agenda - Célébrez chaque petite victoire

3. Utilisez les statistiques - Repérez vos jours les plus productifs - Adaptez votre emploi du temps en conséquence - Partagez vos réussites pour vous motiver

4. Interagissez avec la communauté - Posez des questions dans les forums - Aidez les autres apprenants - Partagez vos expériences

5. Téléchargez les ressources - Sauvegardez les documents importants - Créez des dossiers par module - Consultez-les même hors ligne

Que faire si vous rencontrez des problèmes ?

« Je ne vois pas ma progression » - Rafraîchissez la page (F5) - Vérifiez que vous êtes bien connecté - Contactez le support si le problème persiste

« Mon temps d’apprentissage n’apparaît pas » - Assurez-vous d’avoir regardé des vidéos complètes - Vérifiez que vous avez passé les quiz - Le temps se met à jour toutes les heures

« Je veux changer mes objectifs » - Allez dans « Mon Apprentissage » - Cliquez sur « Mes objectifs » - Modifiez vos buts et sauvegardez

« Je ne comprends pas les statistiques » - Consultez le guide « Comment lire vos stats » - Posez la question dans le forum - Contactez un formateur pour explication

Prochaines étapes recommandées

Après avoir visité votre tableau de bord :

  1. Explorez les cours : Découvrez les 8 modules disponibles

  2. Commencez votre premier module : Le plus souvent « Introduction au droit pénal »

  3. Planifiez votre temps : Fixez-vous des moments réguliers pour étudier

  4. Rejoignez la communauté : Participez aux forums et discussions

Votre parcours d’apprentissage commence maintenant !

Votre tableau de bord est votre porte d’entrée vers une formation de qualité. Utilisez-le chaque jour pour rester motivé et suivre vos progrès.

Chaque session vous rapproche de votre objectif de devenir un expert de la chaîne pénale camerounaise !
